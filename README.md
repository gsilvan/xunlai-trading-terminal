# Xunlai Trading Terminal

Turn Kamadan Trade Chat (https://kamadan.gwtoolbox.com/) into a Xunlai Trading Terminal using Violentmonkey/Tampermonkey.

## Preview
![screenshot](xunlai-trading-terminal.png)

## Installation

1. Install Violentmonkey or Tampermonkey
2. Add script via URL: `https://raw.githubusercontent.com/gsilvan/xunlai-trading-terminal/refs/heads/main/xunlai-trading-terminal.js` or copy and paste its content into a new script.
